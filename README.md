# 🏢 וועד בית — אפליקציית ניהול

אפליקציית PWA לניהול וועד בית — עובדת על מובייל ומחשב.

## תכונות
- ניהול מספר בניינים
- מעקב תשלומי וועד לפי דירה וחודש
- יומן הכנסות והוצאות
- אנשי קשר (דיירים, בעלי דירות, נותני שירות)
- מצב כהה / בהיר
- שמירה מקומית (localStorage)
- ייצוא/ייבוא גיבוי JSON

## שימוש
פתח את `vaad_bait.html` בדפדפן — לא נדרשת התקנה.
